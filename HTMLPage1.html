Project description
  • Get the API of a map and implement it in the page.
  • Define the CTA sectors and show them on the map.
  • Generate random flights:
    • Create a function that generates a random number of flights that fly randodomly from one point to another. Depending on the distance between the points the trajectory will be either a loxodrome or an orthodrome.
    • Create a function that animates the points representing the aircraft for a better visualisation of the simulation.
  • Create the function that counts the aircraft that are inside the area-of-interest. Display the instantaneous number and the total number of aircraft that have passed thorugh it.
Further details will be added to each part as we progress...
